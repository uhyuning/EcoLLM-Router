[ EcoLLM-Router — 프로젝트 루트 ]

프롬프트 복잡도를 실시간으로 분석하여 비용·속도에 최적화된 LLM 모델로
자동 분기(라우팅)하는 지능형 경량화 LLM 라우팅 시스템.

─────────────────────────────────────────────
폴더 구조
─────────────────────────────────────────────
app/          FastAPI 백엔드 서버 (담당: 하윤)
classifier/   프롬프트 난이도 분류 ML 모델 (담당: 유현)
benchmarks/   비용·지연 시간 정량 측정 스크립트
tests/        단위 테스트
docs/         시스템 구조 문서

─────────────────────────────────────────────
핵심 흐름
─────────────────────────────────────────────
사용자 프롬프트
  → classifier: 복잡도 점수 산출 (0.0 ~ 1.0)
  → router: 점수 < 0.5 → Gemini Flash / 점수 ≥ 0.5 → Gemini Pro
  → llm_client: API 호출 및 응답 반환
